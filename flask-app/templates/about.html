About

My name is Niccolo' Ronchetti.
This (work in progress) Flask app is a personal project, result of my passion for TV shows and NLP.
The main goal of this project is to see how good an unsupervised recommender system can become.
We scrape the IMDB database for short synapses of TV shows (in English).
Then, we embed each one of those synapses into a high-dimensional vector, using an advanced NLP model.
We ask the user to enter a short description of waht they'd like to watch. The description is then embedded using the same NLP model, and compared with out offline database of embeddings. We then output the TV Shows having to the synapses most similar to the description entered by the user.

At the time of writing (July 2022), we are using off-the shelf Distilbert tokenizer and model from the Huggingface library.
The notebooks, code and all accompanying information can be found at the link https://github.com/niccronc/TV-Show-Recommender
